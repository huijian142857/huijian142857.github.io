项目规范相关

项目代码部分：

创建 Assembly Definition File 可以把当前目录作为一个程序集，变成 dll 文件，这样通过多个程序集来减少编译时间，如果程序集 A 和 B 互相引用，那么应该把 A 和 B 合为一个程序集。

还有一种方法是，单独建立项目工程，拆分成几个模块，单独编译为 dll 供 unity 使用。
